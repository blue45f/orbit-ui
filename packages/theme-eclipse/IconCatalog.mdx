# Icon Catalog 🪐

Orbit UI에서 제공하는 전체 아이콘 목록입니다. 모든 아이콘은 `size`와 `color` 속성을 지원합니다.

```tsx
import { HomeLineIcon } from '@orbit-ui/icons';

<HomeLineIcon size={24} color="primary" />
```

<br />

<IconGallery>
  {Object.entries(Icons)
    .filter(([name]) => name.endsWith('Icon'))
    .sort(([a], [b]) => a.localeCompare(b))
    .map(([name, Icon]) => (
      <IconItem name={name} key={name}>
        <Icon size={24} />
      </IconItem>
    ))}
</IconGallery>
