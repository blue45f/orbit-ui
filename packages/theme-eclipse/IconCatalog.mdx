# 아이콘 카탈로그

별도 엔트리 포인트로 제공되는 아이콘 목록이에요.  
기본 예제 코드는 다음과 같아요.

```tsx
import { HomeLineIcon } from '@orbit-ui/themes/icons'

const App = () => {
  return <HomeLineIcon size={24} color='fg_primary' />
}
```

제공 중인 아이콘 목록은 [공식문서](https://atelier.claykit.com/docs/foundations/mint/iconography/overview#library)를 참고해주세요.
