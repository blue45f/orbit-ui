# Orbit UI - Component Overview 🪐

Orbit UI는 현대적인 감각과 높은 유연성을 가진 컴포넌트 라이브러리입니다.

## 🎨 Modern UI Patterns

### 1. Soft Shadows & Elevation
모던한 인터페이스를 위해 다층적인 그림자(Layered Shadows)를 사용합니다. 단순히 짙은 그림자가 아닌, 빛의 물리적 특성을 고려한 부드러운 전이를 제공합니다.

### 2. Micro-Interactions
버튼 클릭, 토글 스위치 전환, 모달 열림 등 모든 상호작용에 미세한 애니메이션을 적용하여 사용자에게 생동감 있는 피드백을 전달합니다.

### 3. Glassmorphism & Translucency
배경과 콘텐츠의 깊이감을 위해 투명도와 블러 효과가 적용된 `SurfaceGlass` 토큰을 지원합니다.

## 🏗 핵심 컴포넌트

| 컴포넌트 | 카테고리 | 설명 |
| :--- | :--- | :--- |
| **Buttons** | Actions | Primary, Secondary, Ghost 등 다양한 타입 지원 |
| **Inputs** | Inputs | 텍스트 필드, 검색바, 비밀번호 필드 등 |
| **DataTable** | Data Display | 정렬, 선택, 페이지네이션이 내장된 고성능 테이블 |
| **Feedback** | Feedback | Toast, Modal, Dialog 등 상황에 맞는 알림 |

## 🚀 디자인 가이드라인

- **8px Grid**: 모든 간격과 크기는 8px 단위(0.5rem)로 설계되었습니다.
- **Type-safe**: 모든 컴포넌트는 TypeScript로 작성되어 강력한 타입 추론을 제공합니다.
- **Accessible**: WCAG AA 등급 준수를 목표로 설계되었습니다.
