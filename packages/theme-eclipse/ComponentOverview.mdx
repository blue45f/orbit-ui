# 컴포넌트 목록

`@orbit-ui/theme-eclipse`에서 제공하는 컴포넌트 목록입니다.

<br />

## Buttons

| 컴포넌트 | 설명 |
|----------|------|
| `SolidButton` | 배경이 채워진 기본 버튼 |
| `OutlineButton` | 테두리만 있는 아웃라인 버튼 |
| `GhostButton` | 배경과 테두리가 없는 텍스트 버튼 |
| `SolidIconButton` | 아이콘 전용 채워진 버튼 |
| `OutlineIconButton` | 아이콘 전용 아웃라인 버튼 |

```tsx
import { SolidButton, OutlineButton, GhostButton } from '@orbit-ui/theme-eclipse'
import { SolidIconButton, OutlineIconButton } from '@orbit-ui/theme-eclipse'
```

<br />

## Form Controls

| 컴포넌트 | 설명 |
|----------|------|
| `TextField` | 텍스트 입력 필드 |
| `FloatingTextField` | 플로팅 라벨이 있는 텍스트 필드 |
| `TextArea` | 여러 줄 텍스트 입력 |
| `PasswordField` | 비밀번호 입력 필드 |
| `SearchBar` | 검색 입력 바 |
| `Checkbox` | 체크박스 |
| `BoxedCheckbox` | 박스형 체크박스 |
| `RadioButton` | 라디오 버튼 |
| `Toggle` | 토글 스위치 |
| `Dropdown` | 드롭다운 선택 |

```tsx
import { TextField, TextArea, Checkbox, RadioButton, Toggle } from '@orbit-ui/theme-eclipse'
import { FloatingTextField, PasswordField, SearchBar } from '@orbit-ui/theme-eclipse'
import { BoxedCheckbox, Dropdown } from '@orbit-ui/theme-eclipse'
```

<br />

## Navigation

| 컴포넌트 | 설명 |
|----------|------|
| `AppBar` | 상단 앱 바 |
| `TabGroup` | 탭 그룹 |
| `TabItem` | 탭 아이템 |
| `ScrollableTabGroup` | 스크롤 가능한 탭 그룹 |
| `SegmentedControl` | 세그먼트 컨트롤 |
| `PageDots` | 페이지 도트 인디케이터 |
| `PageIndicator` | 페이지 인디케이터 |
| `PageNumber` | 페이지 번호 인디케이터 |

```tsx
import { AppBar, TabGroup, TabItem, ScrollableTabGroup } from '@orbit-ui/theme-eclipse'
import { SegmentedControl } from '@orbit-ui/theme-eclipse'
import { PageDots, PageIndicator, PageNumber } from '@orbit-ui/theme-eclipse'
```
