# Orbit UI - Eclipse Theme 🪐

**@orbit-ui/theme-eclipse** `v2.0.0-beta.16`

Orbit UI는 현대적인 웹 개발 환경을 위해 설계된 고성능 React 디자인 시스템 컴포넌트 라이브러리입니다. 
Eclipse 테마는 광활한 우주의 깊이감과 정교한 인터랙션을 디자인 언어로 승화시킨 공식 테마 패키지입니다.

## 🏛 아키텍처 및 확장성 (Extensibility)

Orbit UI는 **Headless & Tiered 아키텍처**를 기반으로 하여 디자인 변경과 기능 확장이 매우 유연합니다.

### 1. 슬롯 패턴 (Slot Pattern)
대부분의 컴포넌트는 Leading, Center, Trailing과 같은 서브 컴포넌트를 제공합니다. 이를 통해 컴포넌트 내부의 로직을 건드리지 않고도 원하는 아이콘이나 요소를 자유롭게 주입할 수 있습니다.

```tsx
<SolidButton>
  <SolidButton.Leading><CustomIcon /></SolidButton.Leading>
  <SolidButton.Center>커스텀 버튼</SolidButton.Center>
</SolidButton>
```

### 2. 다형성 (Polymorphic)
as prop을 지원하여 시각적 스타일은 유지하되 실제 렌더링되는 HTML 태그를 변경할 수 있습니다. (예: Button 스타일을 가진 a 태그)

### 3. 테마 및 밀도 제어
EclipseProvider를 통해 전역적으로 테마와 정보 밀도를 제어할 수 있습니다.

| Prop | 값 | 설명 |
| :--- | :--- | :--- |
| mode | light / dark | 라이트/다크 모드 전환 |
| density | standard / high | 일반/고밀도(어드민용) 여백 조정 |
| platform | mobile / pc | 플랫폼별 최적화 스타일 |

## 🌟 디자인 철학 (Premium UX)

Orbit UI는 단순한 시각적 요소를 넘어, 사용자와 제품 사이의 감각적 연결을 중요시합니다.

- **Layered Elevation**: 단순히 짙은 그림자가 아닌, 다층적인 Elevation 시스템을 통해 컴포넌트 간의 계층을 우아하게 분리합니다.
- **Haptic-like Feedback**: 버튼 클릭 시의 미세한 스케일 변화, 토글의 탄성 있는 움직임 등 물리적인 피드백을 디지털로 구현합니다.
- **Visual Consistency**: 모든 아이콘, 컬러, 간격 토큰은 엄격한 그리드 시스템 하에 관리되어 어떤 조합에서도 조화로운 UI를 제공합니다.
