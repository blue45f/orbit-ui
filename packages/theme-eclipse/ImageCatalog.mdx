## 🏗️ 이미지 사용 가이드

### 1. 이미지 경로 규칙
이미지 파일명은 `{이미지 이름}@{스케일}.{확장자}`로 구성됩니다. (예: `welcome@2x.png`)

### 2. 컴포넌트 내 사용 예시
```tsx
const MyComponent = () => {
  return (
    <img 
      src="https://cdn.orbit-ui.com/assets/welcome@2x.png" 
      alt="Welcome to Orbit UI" 
      style={{ width: '400px', height: 'auto' }} 
    />
  )
}
```

### 3. 배포 및 업데이트
현재 이미지 에셋은 CDN 인프라 구축 중이며, 완료 전까지는 위 샘플 이미지를 플레이스홀더로 사용하실 수 있습니다.
