# 이미지 카탈로그

CDN으로 제공되는 이미지 목록이에요.

이미지 파일명은 `{이미지 이름}{스케일}.{확장자}`로 구성돼요.
이클립스는 1x, 2x, 3x 스케일과 png 확장자로 제공될 예정입니다.

현재 이미지 CDN 서비스는 준비 중입니다. 배포가 완료되면 관련 정보를 업데이트할 예정입니다.

제공될 이미지 목록과 상세 가이드는 [공식 GitHub 저장소](https://github.com/blue45f/ui-forge)를 참고해주세요.
