# 이미지 카탈로그

CDN으로 제공되는 이미지 목록이에요.

이미지 파일명은 `{이미지 이름}{스케일}.{확장자}`로 구성돼요.
민트는 1x, 2x, 3x 스케일과 png 확장자로 제공해요.

e.g. 이미지 이름이 logo_service_full_gift인 경우,

- `logo_service_full_gift.png`
- `logo_service_full_gift@2x.png`
- `logo_service_full_gift@3x.png`

환경별 CDN 도메인과 이미지 경로는 아래와 같아요.

- 베타: `https://atelier-assets.betaclaykit.in/images/mint`
- 운영: `https://atelier-assets.claykit.com/images/mint`

제공 중인 이미지 목록은 [공식문서](https://atelier.claykit.com/docs/foundations/mint/image/overview#library)를 참고해주세요.
