## Design Context

### Users
- **오픈소스 디자인 시스템** — 누구나 가져다 쓸 수 있는 범용 React 컴포넌트 라이브러리
- 프론트엔드 개발자가 소비자, 최종 사용자는 모바일 웹뷰 + 데스크톱 웹
- 특정 브랜드에 종속되지 않는 중립적이고 범용적인 디자인
- 코드는 최대한 단순하고 경량하게, 외부 라이브러리(Radix UI) 적극 활용

### Brand Personality
- **Clean** — 불필요한 장식 없이 깔끔함
- **Modern** — 2024-2025 트렌드를 반영한 세련된 느낌
- **Versatile** — 어떤 프로젝트에도 자연스럽게 어울림

### Aesthetic Direction
- **Visual tone**: Minimal, neutral, polished — Vercel/Linear/Radix Themes 수준의 세련된 중립 디자인
- **Color**: 중립 그레이 기반 + 차분한 블루 계열 Primary (과도한 민트/네온 지양)
- **Typography**: SF Pro / Inter / system-ui 기반, 깔끔한 weight hierarchy
- **Spacing**: 4px 기반 consistent spacing scale
- **Radius**: 부드러운 라운딩 (8-12px), 과도한 pill shape 지양
- **Shadows**: 미세한 elevation, 은은한 그림자
- **Theme**: Light mode 기본, Dark mode 지원
- **Anti-references**: 과도한 그라데이션, 네온 색상, glassmorphism, 과한 애니메이션

### Design Principles
1. **Neutral by default** — 특정 브랜드 색상에 의존하지 않고 어디서든 쓸 수 있는 중립적 디자인
2. **Substance over decoration** — 기능이 디자인을 이끌고, 장식은 최소화
3. **Consistent rhythm** — 일관된 spacing, sizing, radius로 시각적 통일감
4. **Accessible contrast** — WCAG AA 이상의 색상 대비, 명확한 상태 표현
5. **Theme-ready** — CSS 변수 기반으로 소비자가 쉽게 커스터마이징 가능
